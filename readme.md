# Fitting Demo

一个可拓展的拟合框架。demo中展示了[双载流子模型的拟合策略](https://pubs.aip.org/jap/article/137/2/025702/3329880/Two-carrier-model-fitting-of-Hall-effect-in)。

## 如何运行本Demo
1. 安装uv
```bash
# 通过pip安装
pip install uv
# 如果python脚本没有add to path，就需要运行：
python -m pip install uv
```
2. 在当前项目路径，运行
```
uv sync
uv run main.py
# 如果python脚本没有add to path，就需要运行：
python -m uv sync
python -m uv run main.py
```
